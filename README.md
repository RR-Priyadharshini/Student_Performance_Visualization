Student Performance Visualization

This project is part of the **Infosys Springboard Virtual Internship (Milestone 1: Exploratory Data Analysis)**.  
The goal is to analyze and visualize factors affecting student performance such as study hours, sleep hours, diet quality, and extracurricular activities.

Dataset Description:
The dataset contains information about 5 students, including:
- Age  
- Gender  
- Diet quality  
- Sleep hours  
- Part-time job  
- Study hours  
- Extracurricular participation  
- Exam score  

Analysis Performed:
- Checked for missing and duplicate values  
- Generated summary statistics  
- Visualized relationships between factors (study hours, sleep, diet) and exam scores  
- Derived insights and conclusions

 Key Insights:
- No missing or duplicate data were found.  
- Most students scored between 70–90 marks.  
- Students with good diet quality, 6–8 hours of sleep, and regular study performed better.

Conclusion:
Healthy habits such as good diet, sufficient sleep, and consistent study help improve academic performance.

 Tools Used
- Python  
- Pandas  
- Matplotlib / Seaborn  
- Google Colab
